Azure

Week 1:

  • Read and understand the concept of cloud computing, including the shared responsibility model and cloud models.
  • Identify appropriate use cases for each cloud model.
  • Compare cloud pricing models and consumption-based model.
  • Study the benefits of using cloud services and the benefits of high availability and scalability in the cloud.

Week 2:

  • Study the benefits of reliability and predictability in the cloud, security, and governance in the cloud, and manageability in the cloud.
  • Understand the different types of cloud services, including IaaS, PaaS, and SaaS.
  • Identify appropriate use cases for each cloud service.

Week 3:

  • Study the core architectural components of Azure, including regional pairs, sovereign regions, availability zones, and data centers.
  • Understand Azure resources and resource groups, subscriptions, and management groups.

Week 4:

  • Study Azure compute and networking services, including compute types, VM options, application hosting options, and virtual networking.
  • Understand Azure storage services, including storage tiers, redundancy options, and storage account options.
  • Study Azure identity, access, and security, including directory services, authentication methods, conditional access, and role-based access control.
  • Study Azure management and governance, including cost management, features and tools for governance and compliance, features and tools for managing and deploying Azure resources, and monitoring tools in Azure.
